El header es el encabezado de un documento o seccion

El nav es la etiqueta utilizada para representar la navegacion de una pagina, como puede ser un menu

El article sirve para desarrollar un contenido completo e independiente

El aside se utiliza para incluir contenido adicional no necesariamente relacionado con lo tratado en la pagina. Puede ser utilizado para incluir banners publicitarios.

En el footer incluiremos el pie de pagina de un documento o seccion.
